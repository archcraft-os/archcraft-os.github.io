Features Overview

Here's a list of main features and functionality added to Archcraft OS.

01

BIOS & UEFI Support

Archcraft can be installed on both BIOS & UEFI system. Works out of the box. Both grub and systemd-boot configured for UEFI boot process. choise is yours however.

02

Offline Installer

Archcraft uses ABIF (Arch Base Installation Framework) as it's installer. It's a generic offline installer for Arch-based ISOs with a lots of features like, BIOS and UEFI Support, Multiple languages support, LUKS (LUKS, LUKS on LVM, and LVM on LUKS), LVM, btrfs (including special mounting), Multiuser & Built-in error detection. Archcraft can be installed on Hard-drive, USB Pendrive, Sdcard, Virtualbox, VMware, etc.

03

Custom Desktop Environment

Archcraft has no desktop environment. it's uses Openbox and Bspwm, both are window managers. Each window manager is highly configured, gives you a feel like a whole desktop environment. There are multiple styles for each WM, which change everything (BG, Panel, Menus, Theme & icons, Fonts Etc) with just one click. It's totally on you which styles you prefer.

04

File Manager & Network Functionality

Archcraft has pre-configured Network-Manager And provides you the full file manager functionalities like Display and access other partitions, removable media (e.g. USB sticks, optical discs, and digital cameras), networking / shared networks, thumbnailing, Archive and extract compressed files, etc. Well sounds like no big deal, maybe for a newbie arch user.

05

Lightweight Applications

Archcraft comes with some lightweight applications, like Thunar(File Manager), Midori(Web Browser), Termite(Terminal), Geany & Leafpad(IDE & Text Editor), Viewnior(Image Viewer), Mplayer(Media Player). Rest is up to users what softwares they want to use on their system. Archcraft is fully capable of running any kind of software.

06

A lot of Useful Tools

Archcraft has some tools and utilities pre-installed. shell applications like htop & bashtop(system monitors), ranger(cli fm), vim & nano(cli text editors), mpd & ncmpcpp(cli music player), zsh shell with OMZ framework & a lots of scripts and tools.
