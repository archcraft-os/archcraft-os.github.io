Things To Do After Installing Archcraft OS

After installing Archcraft OS, there are few things you should do.

  • Update your installation.

$ sudo pacman -Syy && sudo pacman -Syu
  • Install new softwares - install your favorite programs with pacman or yay (AUR).

$ sudo pacman -S firefox gimp

# Or from AUR

$ yay -S spotify
  • Fix Icons - Some icons not showing in the bar.

# Run the following script to fix battery and network modules in polybar

$ ~/.config/polybar/fix_modules.sh
  • Remove unnecessary modules form initrd - Let's say you've installed archcraft on a machine with Intel GPU. In this case you only need i915. Delete rest modules from the array and rebuild initrd.

# Edit mkinitcpio.conf file
$ sudo vim /etc/mkinitcpio.conf

# Change line 7 from
MODULES=(i915? amdgpu? radeon? nouveau? vboxvideo? vmwgfx?)
# To
MODULES=(i915)

# Save file and rebuild initrd
$ sudo mkinitcpio -p linux
  • If you've installed Archcraft without encryption, there are some settings that'll be useless for you in this case. So after normal installation, Do following - 

# Edit grub settings file
$ sudo vim /etc/default/grub

# Comment the following line (13)
GRUB_ENABLE_CRYPTODISK=y

# Update grub config
$ sudo grub-mkconfig -o /boot/grub/grub.cfg

# Remove plymouth-encrypt from hooks
$ sudo vim /etc/mkinitcpio.conf

# Change this line (52) from
HOOKS=(base udev plymouth plymouth-encrypt modconf block filesystems keyboard fsck)
# To
HOOKS=(base udev plymouth autodetect modconf block filesystems keyboard fsck)

# Save file and rebuild initrd
$ sudo mkinitcpio -p linux
  • By default, Touchpad is disabled. To enable it, Press Window or Alt + F1 key to open launcher, Type Mouse & open Mouse and Touchpad. Then goto Touchpad & check Tap to click.

Well, that's pretty much everything for now, go ahead and try out archcraft.
